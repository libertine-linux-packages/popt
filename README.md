## Upstream

* popt is part of the rpm5 project, and is stored in a CVS repository
* It can be obtained using:-

```bash
cvs -d :pserver:anonymous@rpm5.org:/cvs login
cvs -d :pserver:anonymous@rpm5.org:/cvs co popt
```
